## Installation
To get your developemnt env up and running you simply need to do the following steps: 

```bash
git clone <this repo>

virtualenv env
source env\Scripts\activate (if using windows remove the source)
pip install -r requirements.txt

set FLASK_APP=server.py
flask run
```
